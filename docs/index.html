🚀 Quick Start

# Clone the repo
git clone https://github.com/demo112/Try_make_mcp.git
# Install dependencies
pip install -r requirements.txt
# Run the MCP factory
python src/mcp_factory/run.py
# Or use with Claude Desktop
# Add to claude_desktop_config.json
