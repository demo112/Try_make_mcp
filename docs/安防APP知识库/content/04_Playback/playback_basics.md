# 录像回放基础

## 简介
查看存储在 SD 卡或云端的历史录像。

## 操作步骤
1. 在预览页面点击“回放”按钮，或在设备列表页点击“回放”。
2. **选择日期**: 点击日期栏，有圆点标记的日期表示有录像。
3. **时间轴操作**:
   - 蓝色条：表示连续录像。
   - 橙色条：表示报警/事件录像。
   - 左右拖动时间轴寻找目标时间点。
   - 双指开合可缩放时间轴精度。
4. **倍速播放**: 点击倍速图标，支持 2x/4x/8x 快进（部分设备支持）。

## 常见问题
- **Q: 显示“无录像文件”？**
  A: 
  1. 确认 SD 卡已插入并格式化正常（状态为“正常”）。
  2. 确认已开启录像计划（全天录像或移动侦测录像）。
