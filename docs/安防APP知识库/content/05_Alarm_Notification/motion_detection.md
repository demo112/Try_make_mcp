# 移动侦测设置

## 简介
配置摄像机在检测到画面变化时触发报警。

## 操作步骤
1. 进入“设备设置 > 报警设置”。
2. 开启“移动侦测”。
3. **灵敏度设置**:
   - 低：只有大幅度动作才会触发。
   - 中：默认设置。
   - 高：树叶晃动也可能触发（易误报）。
4. **侦测区域**:
   - 点击进入画面网格。
   - 涂抹不需要检测的区域（如街道、摆动的植物）。
   - 只保留需要重点关注的区域（如门口、窗户）。

## 高级功能
- **人形检测**: 开启后，系统只在识别到人形轮廓时才报警，过滤飞虫、光线变化引起的误报。

## 常见问题
- **Q: 误报太多？**
  A: 降低灵敏度，或开启“人形检测”，并重新划定侦测区域。
