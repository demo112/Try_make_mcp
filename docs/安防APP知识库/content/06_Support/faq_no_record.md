# 无录像问题排查

## 现象描述
进入回放界面，时间轴为空白，或提示“无录像文件”。

## 排查步骤
1. **检查 SD 卡状态**: 进入“设置 > 存储设置”，确认状态为“正常”。如果是“异常”或“未格式化”，请先格式化。
2. **检查录像开关**:
   - 确认“全天录像”或“事件录像”开关已打开。
3. **检查录像计划**: 确认当前时间段在设定的录像计划范围内（默认是 7x24 小时）。
4. **检查时区**: 确认设备时区设置正确（如 UTC+8 北京时间）。如果时区错误，录像可能记录到了错误的时间段。

## 特殊情况
- 如果开启了“仅事件录像”，且画面静止无变化，则不会产生录像，这是正常现象（节省空间）。
