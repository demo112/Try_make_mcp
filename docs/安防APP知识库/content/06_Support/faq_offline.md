# 设备离线排查

## 现象描述
APP 上设备显示灰色“离线”状态，无法预览。

## 排查步骤
1. **检查电源**: 确认设备指示灯是否亮起。如果不亮，请检查电源插头。
2. **检查 Wi-Fi**:
   - 路由器是否更换了密码或名称？如果是，需要重置设备重新添加。
   - 设备距离路由器是否太远？尝试移近测试。
3. **重启设备**: 断电重启，观察指示灯状态。
   - 红灯常亮：启动中。
   - 蓝灯/绿灯闪烁：连接网络中。
   - 蓝灯/绿灯常亮：在线。
4. **检查路由器**: 是否连接设备过多导致 IP 冲突。重启路由器尝试。

## 常见问题
- **Q: 偶尔离线又自动上线？**
  A: 通常是 Wi-Fi 信号不稳定导致。建议使用 Wi-Fi 信号增强器或调整路由器位置。
