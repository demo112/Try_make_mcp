# 实时预览基础

## 简介
实时预览是 APP 的核心功能，您可以随时随地查看监控画面。

## 界面说明
进入预览页面后，底部工具栏包含以下功能：
1. **暂停/播放**: 停止或恢复拉流。
2. **声音**: 开启或关闭现场声音监听。
3. **多画面**: 切换 1/4/9/16 分屏模式。
4. **清晰度**: 切换“高清 (HD)”或“标清 (SD)”。
   - *提示: 网络状况不佳时建议切换为标清流畅模式。*
5. **截图**: 抓拍当前画面并保存到手机相册。
6. **本地录像**: 开始录制一段视频保存到手机相册。

## 操作步骤
1. 在首页点击设备缩略图进入预览。
2. 双击屏幕可放大局部细节（电子放大）。
3. 左右滑动可切换到下一个设备。

## 常见问题
- **Q: 画面卡顿或延迟？**
  A: 请检查手机网络信号，或降低清晰度至“标清”。
