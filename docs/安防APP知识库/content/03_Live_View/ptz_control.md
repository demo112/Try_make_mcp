# 云台控制 (PTZ)

## 简介
对于支持云台功能的摄像机，您可以远程控制镜头旋转和变焦。

## 操作步骤
1. 进入实时预览页面。
2. 点击底部工具栏的“云台”图标。
3. **方向控制**: 使用屏幕中间的圆盘上下左右滑动，控制镜头转动。
4. **变焦控制**: 点击放大/缩小图标（如果设备支持光学变焦）。
5. **预置点**:
   - 设置：将镜头移至目标位置，点击“添加预置点”，输入名称。
   - 调用：点击已保存的预置点，镜头自动转至该位置。

## 常见问题
- **Q: 找不到云台图标？**
  A: 请确认您的设备型号是否支持云台功能。
- **Q: 控制有延迟？**
  A: 云台指令通过网络传输，会有 1-3 秒的延迟，属正常现象。
