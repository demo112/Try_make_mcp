# IP/域名添加设备

## 简介
适用于通过公网 IP 或域名访问的设备，常用于 NVR 或专业摄像机。

## 操作步骤
1. 点击 APP 首页右上角的“+”号。
2. 选择“手动添加”。
3. 切换添加方式为“IP/Domain”。
4. 填写以下信息：
   - **别名**: 自定义（如“公司仓库”）
   - **地址**: 设备的公网 IP 或域名
   - **端口**: 服务端口（默认 8000）
   - **用户名**: 设备管理员账号（默认 admin）
   - **密码**: 设备登录密码
5. 点击右上角保存图标。

## 常见问题
- **Q: 添加失败？**
  A: 请检查 IP 是否可达，端口是否已在路由器做映射，以及账号密码是否正确。
