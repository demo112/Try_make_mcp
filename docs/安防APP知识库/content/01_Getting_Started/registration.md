# 账号注册

## 简介
使用安防APP前，您需要注册一个个人账号，以便管理您的设备。

## 操作步骤
1. 打开安防APP，点击登录页面的“新用户注册”。
2. 选择您所在的国家/地区（**注意：注册后不可更改**）。
3. 输入您的手机号码或电子邮箱。
4. 点击“获取验证码”，输入收到的 6 位验证码。
5. 设置登录密码（需包含字母和数字，8-16位）。
6. 点击“完成注册”，系统将自动登录。

## 常见问题
- **Q: 收不到验证码？**
  A: 请检查手机是否拦截了短信，或尝试等待 60 秒后重新获取。
- **Q: 提示账号已存在？**
  A: 请直接尝试登录，或使用“忘记密码”功能找回。
