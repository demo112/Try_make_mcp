# APP安装指南

## 简介
本文档指导用户如何下载并安装安防APP到您的移动设备。

## 前提条件
- 智能手机（iOS 10.0+ 或 Android 5.0+）
- 网络连接正常

## 操作步骤

### 1. 扫描二维码下载
1. 打开手机相机或微信“扫一扫”。
2. 扫描设备包装盒或说明书上的 APP 下载二维码。
3. 根据提示跳转至 App Store (iOS) 或 Google Play/应用宝 (Android) 进行下载。

### 2. 应用商店搜索下载
1. 打开手机应用商店。
2. 在搜索框输入“安防卫士”（示例名称）。
3. 点击“获取”或“安装”按钮。

## 常见问题
- **Q: 扫描二维码无反应？**
  A: 请检查网络连接，或尝试直接在应用商店搜索下载。
- **Q: 安装失败提示“解析包错误”？**
  A: Android 用户请确保已开启“允许安装未知来源应用”权限，或重新下载安装包。
