# ALIGNMENT_安防APP知识库

## 1. 背景 (Context)
用户希望在 ragflow 服务中构建一个“安防APP应有的完整的文档知识库”。这是一个文档工程任务，旨在生成高质量、结构化的 Markdown 文档，供 RAG (Retrieval-Augmented Generation) 系统索引和检索。

## 2. 目标 (Goals)
构建一套覆盖安防APP全生命周期的文档集合，包括：
- 产品介绍与快速入门
- 核心功能操作指南
- 常见问题解答 (FAQ)
- 故障排查手册

## 3. 核心需求 (Core Requirements)
知识库需包含以下核心模块：

### 3.1 基础模块
- **APP安装与登录**: iOS/Android 下载渠道，账号注册，忘记密码。
- **设备添加**: 扫码添加，局域网搜索，手动输入序列号。

### 3.2 功能模块
- **实时预览 (Live View)**: 画面切换，云台控制 (PTZ)，双向对讲，截图/录像。
- **录像回放 (Playback)**: 时间轴回放，事件筛选，录像下载/分享。
- **报警通知 (Alarm)**: 消息推送设置，报警类型说明（移动侦测，人形检测）。
- **设备管理**: 设备分享，固件升级，网络配置。

### 3.3 支持模块
- **常见问题 (FAQ)**: 离线问题，推流失败，SD卡未识别。
- **安全隐私**: 数据加密说明，隐私政策摘要。

## 4. 交付物 (Deliverables)
- 一系列 Markdown 文件，存放在 `src/apps/security_kb/data` (假设路径) 或 `docs/安防APP知识库/content` 中。
- 清晰的目录结构。

## 5. 约束 (Constraints)
- 格式：Markdown
- 语言：中文
- 风格：清晰、简洁、以问题解决为导向（适合 RAG 检索）。
